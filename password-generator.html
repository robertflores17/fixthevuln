Password Security Tips

  • Length matters most - Longer passwords are exponentially harder to crack
  • Use unique passwords - Never reuse passwords across sites
  • Use a password manager - Store passwords securely (1Password, Bitwarden, KeePass)
  • Enable MFA - Add a second factor wherever possible
  • Passphrases are memorable - 4+ random words can be both secure and easy to remember
