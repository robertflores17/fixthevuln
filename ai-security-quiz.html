Sample AI Security Practice Questions

Q: What is a direct prompt injection attack?

A: Crafting user input that overrides the system prompt or safety instructions — Direct prompt injection occurs when a user crafts input specifically designed to override, ignore, or bypass the system prompt and safety guardrails.

Q: An attacker embeds hidden instructions in a webpage that an AI agent retrieves during a search task. What type of attack is this?

A: Indirect prompt injection — Indirect prompt injection occurs when malicious instructions are placed in external data sources that an AI agent later retrieves and processes.

Q: Which defense is most effective against direct prompt injection attacks?

A: Input validation combined with output filtering — Input validation detects and sanitizes injection patterns while output filtering blocks sensitive content from responses, providing layered defense.

Q: What is a membership inference attack against an AI model?

A: Determining whether a specific data record was used in the model's training set — Membership inference attacks query the model to determine if a specific individual's data was included in the training dataset, posing privacy risks.

Q: An attacker publishes an AI agent plugin named 'aws-s3-helper' that closely mimics a popular legitimate plugin 'aws-s3-tools'. What supply chain attack technique is this?

A: Typosquatting — Typosquatting creates packages or plugins with names very similar to popular legitimate ones, exploiting common typing mistakes.
