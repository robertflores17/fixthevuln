AZ-900 vs AZ-104

Which certification should you get?

Last updated: 2026-04-21

By FixTheVuln Team · Independent certification guidance

AZ-900 and AZ-104 represent fundamentals and intermediate levels of Azure certification. AZ-900 validates basic cloud concepts and Azure services knowledge, while AZ-104 tests hands-on Azure administration across compute, networking, storage, and identity.

Focus Areas

Microsoft Azure Fundamentals

Cloud concepts, Azure services, security, privacy, compliance, trust, and Azure pricing and support

Microsoft Azure Administrator

Managing Azure identities and governance, implementing and managing storage, deploying and managing Azure compute resources, configuring and managing virtual networking, and monitoring and maintaining Azure resources

Who Should Get Which?

Get Microsoft Azure Fundamentals if...

IT beginners, non-technical professionals, or anyone who wants to demonstrate basic Azure understanding without hands-on administration skills

Get Microsoft Azure Administrator if...

IT administrators managing Azure environments, system admins transitioning to cloud, or anyone who needs to prove hands-on Azure administration competency

Recommended Order

Skip AZ-900 if you have IT experience and go directly to AZ-104. AZ-900 is a fundamentals exam designed for non-technical audiences. If you have any system administration background, AZ-104 is the right starting point.

Study Tips

AZ-900 can be passed with Microsoft Learn free modules in 1-2 weeks. AZ-104 requires hands-on Azure Portal and CLI practice over 4-8 weeks. Focus AZ-104 study on managing virtual machines, virtual networks, storage accounts, and Microsoft Entra ID (Azure AD).
