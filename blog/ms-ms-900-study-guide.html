Microsoft MS-900 Study Guide: Everything You Need to Pass the MS-900 Exam

Expert cybersecurity insights for IT professionals

Last updated: March 1, 2026

By FixTheVuln Team · Peer-reviewed security content · Sources: Industry frameworks and standards

The Microsoft MS-900 certification validates your expertise and opens doors to higher-paying roles in IT and cybersecurity. Whether you are just starting your study journey or doing a final review, this guide breaks down everything you need to know to pass the MS-900 exam.

Exam Overview

  • Certification: Microsoft MS-900
  • Exam Code: MS-900
  • Vendor: Microsoft
  • Cost: $99 USD
  • Duration: 65 minutes
  • Questions: 40-60 questions
  • Passing Score: 700 out of 1000
  • Format: Multiple choice and multiple response
  • Prerequisites: None required

Domain Breakdown

Understanding the exam domains and their weights is critical for efficient study planning. Focus more time on heavily-weighted domains while ensuring you cover all areas.

Domain 1: Describe Cloud Concepts (10-15%)

  • 1.1 Describe the different types of cloud services
  • 1.2 Describe the benefits of using cloud services
  • 1.3 Describe the shared responsibility model

Key concepts: IaaS, PaaS, SaaS, Public/Private/Hybrid Cloud, Shared Responsibility Model, Scalability, Elasticity, High Availability

Domain 2: Describe Microsoft 365 Apps and Services (45-50%)

  • 2.1 Describe Microsoft 365 productivity solutions
  • 2.2 Describe collaboration solutions in Microsoft 365
  • 2.3 Describe endpoint modernization concepts
  • 2.4 Describe analytics capabilities in Microsoft 365

Key concepts: Microsoft Teams, SharePoint Online, OneDrive for Business, Exchange Online, Microsoft Loop, Microsoft Viva, Microsoft Copilot, Power Platform

Domain 3: Describe Security, Compliance, Privacy, and Trust (15-20%)

  • 3.1 Describe identity and access management solutions
  • 3.2 Describe threat protection with Microsoft Defender
  • 3.3 Describe trust, privacy, and compliance solutions

Key concepts: Microsoft Entra ID, Conditional Access, Multi-Factor Authentication, Microsoft Defender for Office 365, Microsoft Defender for Endpoint, Microsoft Intune, Data Loss Prevention, Sensitivity Labels

Domain 4: Describe Microsoft 365 Pricing, Licensing, and Support (15-20%)

  • 4.1 Identify Microsoft 365 pricing and billing management options
  • 4.2 Identify licensing options available in Microsoft 365
  • 4.3 Identify support options for Microsoft 365 services

Key concepts: Microsoft 365 Business Plans, Microsoft 365 Enterprise E3/E5, Microsoft 365 F1/F3 Frontline, Add-On Licenses, Per-User Licensing, Microsoft 365 Admin Center, Service Health Dashboard, Support Ticket Process

Recommended Study Timeline

Plan for approximately 4-6 weeks of dedicated study. Here is a suggested weekly breakdown:

  • Week 1: Cloud Concepts: IaaS/PaaS/SaaS, shared responsibility, cloud benefits
  • Week 2: M365 Apps: Teams, SharePoint, OneDrive, Exchange Online, Outlook
  • Week 3: M365 Apps: Viva, Power Platform, Microsoft Loop, Copilot
  • Week 4: Security: Microsoft Defender, Intune, Entra ID, Conditional Access
  • Week 5: Compliance: Purview, DLP, insider risk, eDiscovery, information protection
  • Week 6: Full Review: Licensing (E3/E5/Business), support options, practice exams

Top Study Tips

  1. Start with the official exam objectives. Download them from the Microsoft website and use them as your study checklist. Every exam question maps to a specific objective.
  1. Use active recall over passive reading. Instead of re-reading notes, test yourself with practice questions after each study session. This dramatically improves retention.
  1. Focus on heavily-weighted domains first. Domains with higher percentages appear more on the exam. Master these before moving to lower-weighted areas.
  1. Build hands-on experience. Set up a lab environment and practice the skills you are studying. Hands-on experience is especially valuable for performance-based questions.
  1. Take practice exams under real conditions. Time yourself, eliminate distractions, and simulate the exam environment. Review every wrong answer and understand why it was wrong.
