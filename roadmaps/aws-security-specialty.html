Week-by-Week Study Plan

Week 1

Domain 1: Threat Detection — GuardDuty findings, CloudWatch anomaly detection, VPC Flow Logs analysis

Week 2

Domain 1: Detection Mechanisms — AWS Config rules, Security Hub, custom detection with Lambda

Week 3

Domain 2: Incident Response — IR plans, compromised EC2/IAM/S3, forensic snapshot isolation

Week 4

Domain 2: Logging & Monitoring — CloudTrail integrity, centralized logging, CloudWatch Logs Insights

Week 5

Domain 3: Edge Security — CloudFront signed URLs, WAF rules, Shield Advanced, DDoS mitigation

Week 6

Domain 3: Network & Compute Security — VPC design, NACLs, security groups, Systems Manager patching

Week 7

Domain 4: IAM Deep Dive — Permission boundaries, SCPs, ABAC, identity federation, SAML/OIDC

Week 8

Domain 4: Advanced IAM — Cross-account roles, IAM Access Analyzer, resource-based policies

Week 9

Domain 5: Data Protection — KMS CMKs, key policies, envelope encryption, Macie, S3 encryption

Week 10

Domain 5: Data in Transit — ACM, TLS termination, VPN encryption, Secrets Manager rotation

Week 11

Domain 6: Governance — AWS Organizations, Control Tower, Config conformance packs, Audit Manager

Week 12

Full Review: Practice exams, Scenario walkthroughs, Weak areas, Exam logistics
